Directional.js

What is this?

Directional.js is a simple utility that gives you access to a direciton varibale as well as emit events that correspond to the direciton that the user is scrolling.

How do I use it?

Very simple, all you have to do is include directional.js after jQuery and instantiate it like this $(window).directional(); That will allow you to use the direction variable plus the up, down, left, right events in your code. wheeeee.

Examples:

if(direction === 'down'){
  //this variable will contain an 
  //up, down, left, right string 
}

$(window).on('up', function(){
  //fires while scrolling up 
});

$(window).on('right', function(){
  //fires while scrolling right 
});

$(window).on('down', function(){
  //fires while scrolling down 
});

$(window).on('left', function(){
  //fires while scrolling left 
});

If you only want the function to run once you can use jQuery's .one(); method.

$(window).one('up', function(){
  //fires one time on an up scroll 
});
